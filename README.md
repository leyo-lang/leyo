# Leyo

Leyo is a compiled programming language, bytecode format, and virtual machine written in C. The project contains a complete toolchain capable of compiling source code into Leyo bytecode, packaging it into a portable file format, executing it inside a virtual machine, and inspecting the generated instructions with a built-in disassembler.

The implementation is intentionally lightweight. Each part of the compiler and runtime is designed to remain understandable without requiring a large framework or external dependencies.

## Design Philosophy

Leyo is built around three core ideas.

### Permissive

The language is intended to give developers control. Rather than hiding implementation details or enforcing unnecessary abstractions, Leyo aims to expose the underlying system while remaining practical to use.

### Observable

Compilation and execution should never be a black box. The toolchain provides visibility into every stage of the process, including generated bytecode, runtime execution, diagnostics, and compiler output.

### Traceable

Logging is a first-class feature throughout the project. Compiler actions, runtime events, controller operations, and errors can all be recorded, making debugging and development significantly easier.

## Features

* Hand-written lexer
* Hand-written parser
* Custom bytecode instruction set
* Portable `.lybc` bytecode format
* Virtual machine runtime
* Built-in bytecode disassembler
* Configurable logging
* Interactive configuration wizard
* Cross-platform C implementation
* Smoke-test framework

## Getting Started

After building Leyo, run:

```bash
leyo help
```

The built-in help system lists every available command together with usage information and examples.

## Main Components

| Component       | Purpose                                                    |
| --------------- | ---------------------------------------------------------- |
| Lexer           | Converts source code into tokens.                          |
| Parser          | Produces Leyo bytecode.                                    |
| Header Writer   | Packages bytecode into the `.lybc` format.                 |
| Runner          | Loads compiled bytecode files.                             |
| Virtual Machine | Executes Leyo bytecode.                                    |
| Disassembler    | Displays bytecode as readable instructions or hexadecimal. |
| Logger          | Records compiler, runtime, controller, and error events.   |
| Configuration   | Loads and manages project settings from `.lyst`.           |
